- `QUEUED`: 큐 대기중
- `PLANNING`: 쿼리 계획중
- `STARTING`: 쿼리 실행 시작
- `RUNNING`: 쿼리 실행중인 Task 존재
- `BLOCKED`: 쿼리가 차단되고 **메모리, 버퍼 공간 대기**
- `FINISHING`: 쿼리가 커밋이 진행중이고 완료되는 상태
- `FINISHED`: 쿼리 실행 완료
- `FAILED`: 쿼리 실행 실패
