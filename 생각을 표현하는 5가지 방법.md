- 데이터 -> 정보 -> 지식 -> **통찰** -> 지혜
- **데이터**: 느슨한 조각들, 그대로 놔두면 쓰레기에 불과함
- **정보**: 데이터를 분류한 것. 어떤 조각들은 x, 어떤 조각들은 y
- **지식**: 경험과 연습을 통해 개별 조각들은 연결한 것
	- 모든 RDB가 foreign key로 지식을 표현함
- **통찰**: 단절된 듯 보이든 정보 조각을 새롭게 연결한 것
- **지혜**: 통찰을 보여주는 서로 다른 노드 간의 특정 경로가 합리적일 때
